Introduccion a TS

Fundamentos históricos

TypeScript fue publicado en octubre de 2012, después de dos años de desarrollo por parte de Microsoft.
Poco después de su publicación, Miguel de Icaza elogió el lenguaje en sí mismo, pero criticó la falta de un IDE maduro para todas las plataformas, en ese momento Microsoft Visual Studio solo estaba disponible para Windows.
En la actualidad existen múltiples editores, como Sublime Text, Vim(El famoso editor de texto plano), Emacs, Webstorm y el propio VSC de Microsoft.

Como crear un proyecto en TS

Gestion de dependencias y config del compilador en ts

Lo primero que haremos sera abrir un aterminal y crearemos un directorio de trabajo

  • mkdir praticats
  • cd practicats

Ahora ejecutaremos el siguiente comando que nos genera un documentos package.json en la raíz de nuestro proyecto, el cual lo usaremos para las dependencias. Luego además instalamos el compilador de TypeScript

  • npm init --yes
  • npm i --global typescript

Ahora crearemos un fichero tsconfig.json y tendrá el siguiente contenido

  • {
  • "compilerOptions": {
  • "target": "ES2024",
  • "module": "commonjs",
  • "rootDir": "./src",
  • "outDir": "./dist"
  • }
  • }

Compilación y ejecución automática

Para compilar y ejecutar haremos esta serie de pasos en los que crearemos un fichero y lo ejecutaremos

  1. mkdir src
  2. cd src/
  3. code hello.ts (escribiemos console.log("hello world"))
  4. cd ..
  5. tsc
  6. node dist/hello.js

De esta form con el tsc compilamos y luego lo que se nos crear en el dir dist/, lo ejecutamos con node

Por otro lado si quieres ir compilando segun vas cambiando el codigo instalaremos esto

  • npm install --save-dev tsc-watch

Y para ejecutar el comando de lo que acabamos de instalar haremos npm start

Tipos de datos en TypeScript

Tipo Descripción Ejemplo
Number Representa números, tanto enteros como decimales let num: number = 5
String Representa cadenas de texto let cadena: string = "Hola"
Boolean Representa valores booleanos (true o false) let verdadero: boolean = true
Array Representa una lista de elementos, que pueden o no ser del mismo tipo let lista: number[] = [1,2,3]
Any Representa cualquier tipo de dato let variable: any = "Hola"
Undefined Representa un valor indefinido let vacio: undefined = undefined
Unión de tipos Permite definir una variable con varios tipos posibles let numero: number | undefined = 5
